# 🔷 Open Hardware Hex Badge

A compact hexagon-shaped open-source hardware badge designed for learning PCB design, electronics, and embedded systems.

This project is part of a hands-on learning journey in hardware development, embedded systems, and robotics exploration.

---
# 🎯 Purpose

This badge is designed to help understand and practice:

- PCB design workflow (schematic → layout → fabrication)
- LED-based embedded hardware systems
- Open-source hardware development process
- Real-world manufacturing preparation using Gerber files
- Basics of electronic product design

The focus is not complexity, but learning the complete hardware creation cycle.

---

# ⚙️ Features

- Hexagon-shaped PCB design
- LED-based visual indicator system
- Clean and minimal silkscreen layout
- Open hardware development approach
- Beginner-friendly PCB project
- Manufacturing-ready design (Gerbers included)
